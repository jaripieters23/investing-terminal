About
AI providerScoring and analysis via Groq (llama-3.3-70b for scoring, llama-3.1-8b for detail pages). Financial data split by free tier: Finnhub (quotes, profiles, financials, metrics, news, insider, recommendations, earnings — 60 req/min, no daily cap) + FMP only for data unavailable free on Finnhub: ROIC/EV·EBITDA key-metrics, price targets, forward estimates (250 req/day, cached 24hr). All via Cloudflare Workers — no API keys in the app.
Quality scoringBased on Pieter Slegers' Compounding Quality. 13 quantitative criteria (rule-based, 60%) + 7 qualitative factors scored by AI (40%). Final score /100.
PrivacyAll portfolio data stays in your browser. API keys are stored securely in your Cloudflare Workers and never touch the app.
