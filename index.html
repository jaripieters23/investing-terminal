About
AI providerAll scoring and qualitative analysis is done by Groq (llama-3.3-70b-versatile for scoring, llama-3.1-8b-instant for detail pages). Financial data: Finnhub (quotes, profiles, financials, news) + FMP (key metrics, analyst estimates, price targets). All calls via Cloudflare Workers — no API keys stored in the app.
Quality scoringBased on Pieter Slegers' Compounding Quality. 13 quantitative criteria (rule-based, 60%) + 7 qualitative factors scored by AI (40%). Final score /100.
PrivacyAll portfolio data stays in your browser. API keys are stored securely in your Cloudflare Workers and never touch the app.
